Practicando github

¿Cómo subir un commit?

Ir a la pagina de github

Registrarse en la pagina web con un correo valido que utilizas actualmente

En el terminal resistrar es mismo correo: 'git config --global user.email "penDavis@chu.com"

En tu pagina de github clicar NEW y se abre una pagina para llenar

Llenar el repository name, y otros esapcios. clicar: Create Repository

Normalmente la siguiente pagina viene activado en HTTPS

Escger de …or push an existing repository from the command line:

git remote add origin https://github.com/renesalen/subir-a-github.git, y pegarlo en el Terminal

Copiamos del mismo: git push -u origin main y pegamos en el terminal, para que el futuro subamos nuestro contenido git a github con solo: 'git push'
